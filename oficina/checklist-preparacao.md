# Checklist de preparação

Fonte canônica da preparação. Marcar os itens conforme forem concluídos.

## Infraestrutura (laboratório)

- [ ] Confirmar laboratório e número de máquinas (define as vagas; decidir 1 máquina por pessoa ou por dupla)
- [ ] VS Code instalado e atualizado em todas as máquinas
- [ ] Git instalado em todas
- [ ] Agente de IA funcionando em todas: decidir entre Copilot (cota gratuita com conta GitHub) e Claude Code com API institucional (falar com o Saimon)
- [ ] Contas GitHub de apoio preparadas e login testado (se Copilot)
- [ ] Repositório da oficina clonado e aberto no VS Code de cada máquina
- [ ] Rodar um setup completo de teste em pelo menos 2 máquinas do laboratório
- [ ] Máquina reserva e plano B de conexão
- [ ] Projetor testado com a máquina do facilitador

## Testes de modelo (pendência da reunião)

- [ ] Testar o ICM de ponta a ponta com o Copilot gratuito (teste informal já feito com a Thammy)
- [ ] Verificar se a cota gratuita aguenta uma oficina inteira por máquina
- [ ] Se API da Anthropic: chave, limites e custo estimado com o Saimon
